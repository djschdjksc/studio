# **App Name**: BillTrack Pro

## Core Features:

- Multi-Panel Search: Enable searching by party name, address, date, slip number, vehicle number/name, and bill type (sale, sale return).
- Dynamic Table: Display a table with columns for Sr. No, Item Name (with alias search), Quantity, Unit, U Cap, and L Cap, with dynamic row and column insertion based on requirements.
- Total Panel: Show a side panel for totals, with columns for Item (grouped by item group), Total Qty (summed group-wise), Price, and Total Price.
- Grand Total: Display a grand total at the bottom of both panels within a border.
- Scrollable Boards: Make all rows and columns scrollable within a border.
- Add New Elements: Add 'New Item' and 'New Party' buttons.
- Item & Party Panels: Manage items with options for item group, unit, and alias code. Manage parties with name, address, and phone number.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and stability.
- Background color: Light gray (#F0F0F0), offering a neutral backdrop that ensures legibility and reduces eye strain.
- Accent color: Soft orange (#FFAB40) for interactive elements like buttons and highlights, providing visual cues without overwhelming the user.
- Font pairing: 'Inter' sans-serif for both body and headlines, offering a clean, modern, and highly readable interface.
- Prioritize clear sectioning and spacing to organize the dense amount of information.
- Use simple, outline-style icons to represent different functions and data points.
- Implement subtle transitions for data updates and panel expansions.
